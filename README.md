
# 💱 Conversor de Moedas

Projeto desenvolvido com HTML, CSS e JavaScript, com o objetivo de realizar conversões entre diferentes moedas de forma simples e intuitiva.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## ⚙️ Funcionalidades

- Conversão entre moedas
- Interface simples e intuitiva
- Atualização dinâmica dos valores
